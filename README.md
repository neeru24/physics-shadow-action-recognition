A research-focused project that explores **human action recognition using shadow dynamics and physics-inspired features**.

## üìå Overview

Human action recognition is traditionally performed using RGB videos, which can be sensitive to lighting, appearance, and background noise.  
This project investigates an alternative approach by focusing on **shadows** as a primary source of information.

Shadows:
- Preserve motion patterns
- Reduce appearance bias
- Improve robustness in privacy-sensitive scenarios
